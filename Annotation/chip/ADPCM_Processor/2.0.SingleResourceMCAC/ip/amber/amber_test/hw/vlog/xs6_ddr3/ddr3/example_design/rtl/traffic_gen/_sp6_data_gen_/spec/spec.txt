Module name: sp6_data_gen. Module specification: The sp6_data_gen module is a data generator designed for system-peripheral (sp6) systems. It generates various data patterns based on input parameters and modes, including walking, hammer, address-based, and pseudorandom binary sequence (PRBS) patterns. The module has inputs for clock (clk_i), reset (rst_i), PRBS seed (prbs_fseed_i), data mode (data_mode_i), data ready (data_rdy_i), command starts (cmd_startA-E), fixed data (fixed_data_i), address (addr_i), user burst count (user_burst_cnt), and FIFO ready (fifo_rdy_i).
