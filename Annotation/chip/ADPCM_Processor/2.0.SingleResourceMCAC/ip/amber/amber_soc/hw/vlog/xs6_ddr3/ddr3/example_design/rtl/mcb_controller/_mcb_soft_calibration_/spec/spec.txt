Module name: mcb_soft_calibration. Module specification: This module performs soft calibration for a memory controller block (MCB), handling initialization and dynamic calibration of I/O termination and DQS delays. It has input ports for clock (UI_CLK), reset (RST), PLL status (PLL_LOCK), self-refresh control (SELFREFRESH_REQ, SELFREFRESH_MCB_MODE), I/O termination calibration (RZQ_IN, RZQ_IODRP_SDO, ZIO_IN, ZIO_IODRP_SDO), and various MCB status and data signals.
