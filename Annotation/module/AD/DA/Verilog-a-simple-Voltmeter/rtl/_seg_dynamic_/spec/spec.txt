Module name: seg_dynamic. Module specification: The `seg_dynamic` module is responsible for driving a seven-segment display based on a 20-bit input representing numerical data. This includes dynamically updating each digit display with decimal to BCD converted values, handling sign representation, and multiplexing the display digits with a refresh rate of 1 millisecond. The input ports include `sys_clk` (system clock), `sys_rst_n` (active low reset), `point` (6-bit decimal point location), `seg_en` (display enable), `data` (20-bit input data), and `sign` (sign of the input data). The output ports are `sel` (6-bit segment selector) and `seg` (8-bit segment data). Internal signals play crucial roles: `unit` to `h_hun` hold BCD digit values; `data_reg` stores the formatted display data; `cnt_1ms` and `flag_1ms` facilitate timing for display refresh; `cnt_sel` selects the active display digit; and `data_disp`, `sel_reg`, `dot_disp` manage the display data, active digit, and decimal points, respectively. Structurally, the code includes blocks for internal state initialization and updates, BCD conversion (via the `bcd_8421` instance), display multiplexing logic, and segment output logic, ensuring the correct visual representation of the input numerical data on the dynamic seven-segment display.
