Module-level comment: The 'key_control' module selects output waveforms (sine, square, triangle, sawtooth) based on the input from four keys. Utilizing the 'sys_clk' for synchronization and 'sys_rst_n' for reset, it updates 'wave_select' with the pressed key's corresponding waveform code. Internal debounce logic via 'key_filter' instances ensures stable input to the waveform selection mechanism.
